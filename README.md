MCCheck是一款利用小天机器人  
查询我的世界服务器人数信息插件  
运行该插件需要安装Java环境   

目前插件仍在测试版，启动前请  
应用中心搜索www  
输入密钥:123按回车添加应用  
到个人中心启动应用  

启动命令  
java -jar MCCheck.jar
